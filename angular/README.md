# Desafio Estágio - Frontend

Frontend application for the Estágio challenge project.

## Development

Run `ng serve` to start the development server. Navigate to `http://localhost:4200/`.

## Build

Run `ng build` to build the project. Build artifacts are stored in `dist/`.
